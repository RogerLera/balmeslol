EstadisticasController

class EstadisticasController extends Controller

Clase EstadisticasController que guardara datos de las partidas contenidas en los ficheros seed_data, estos ficheros contienen datos de 1000 partidas de tipo ranked son ficheros de tipo json, para luego hacer las estadisticas.
